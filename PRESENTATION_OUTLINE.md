## 2. 作品集页面文案

青柠集是一个面向校园学生的闲置交换 App 本地可运行原型，聚焦教材、设计用品、数码用品和宿舍用品等校内二手物品流转场景。项目从用户痛点出发，将商品浏览、搜索筛选、发布编辑、收藏、状态管理、交易记录、联系卖家和聊天详情整合到同一套移动端流程中。

在设计上，青柠集采用清爽的青柠绿视觉风格，强调轻量、校园感和易用性；在实现上，项目不仅包含手机端界面原型，也搭建了 Node.js 本地后端 API，并用 JSON 文件模拟商品和交易记录数据。当前版本不是已上线产品，而是用于课程展示和作品集说明的可运行 Demo，重点呈现产品流程、功能结构和前后端联动能力。

## 3. 汇报讲解稿

大家好，这是我的课程项目“青柠集”，它是一个校园闲置交换 App 的本地可运行原型。项目面向校园学生，主要解决教材、设计用品、数码用品和宿舍用品在校内流转时信息分散、沟通不集中、商品状态不清楚的问题。

在使用流程上，用户打开 App 后可以先在首页浏览商品，也可以通过搜索和分类快速找到目标物品。点击商品后会进入详情页，查看价格、地点、卖家和状态。如果感兴趣，可以收藏商品，也可以联系卖家，系统会生成消息会话，并进入聊天详情进行本地消息发送。

如果用户是卖家，可以通过发布功能新增商品，上传图片时前端会做压缩处理。发布后的商品会出现在“我的发布”里，用户可以继续编辑、删除，或者把状态切换为在售、已预定、已出。状态变化会记录到交易记录中，形成一个基础的商品流转管理闭环。

这个项目不是单纯的静态页面。前端主要由 `mobile.html` 实现，后端使用 Node.js 提供 API，商品和交易记录分别保存在本地 JSON 文件中。因此它可以在本地完整运行，适合做课程演示和作品集展示。不过当前版本仍然是原型阶段，没有真实登录、线上数据库、云端图片存储和真实即时通信。后续如果继续扩展，可以加入校园认证、真实聊天、云端部署和交易评价等功能。
